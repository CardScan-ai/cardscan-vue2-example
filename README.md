# Vue 2, TypeScript, and CardScan SDK Integration

## Introduction

Welcome to the Vue 2, TypeScript, and CardScan SDK integration project! This project seamlessly incorporates the CardScan SDK into a Vue 2 application, leveraging TypeScript for enhanced type checking and Vite for efficient bundling and development.

## Prerequisites

Before diving into the integration, ensure you have the following prerequisites installed on your system:

- **Node.js:** Version 14.x or newer.
- **npm or yarn:** Choose either npm (version 6+) or yarn as your package manager.

## Installation

To get started, follow these simple steps to set up the project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/CardScan-ai/vue-2-cardscan.git
   ```
2. Navigate into the project directory:
   ```bash
   cd vue-2-cardscan
   ```
3. Install project dependencies:
   ```bash
   npm install
   ```

## Running the Project

Once you've installed the necessary dependencies, you're ready to launch the development server. Simply execute the following command:

```bash
npm run dev
```
